# pub-subs

## Main Problem : What if You want to create a system where users can subscribe to the feed of stocks (prices),This application will be used by >1Mn users. How would you build it?
### Create a very basic pub-subs implementation using redis and ts
- [ ] Create a pubsubs manager class 
- [ ] Create an index.ts file to simulate user
- [ ] Create a pubsubs manager class 

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.2.21. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
